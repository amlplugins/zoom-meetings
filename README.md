# @amlplugins/zoom-meetings

Native `@zoom/rest-api` SDK re-exported under the `@amlplugins` namespace.

> Zoom Meetings REST API — meetings, participants, recordings, polls, breakout rooms, registration (Server-to-Server OAuth).

## Install

```bash
npm install @amlplugins/zoom-meetings
```

## Use

```ts
import { /* named exports from @zoom/rest-api */ } from "@amlplugins/zoom-meetings";
// or
import sdk from "@amlplugins/zoom-meetings";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
